# ChatGPT-Puzzle

브라우저에서 바로 실행할 수 있는 4x4 동물 카드 기억력 퍼즐 게임입니다. 같은 동물 카드 두 장을 찾아 모든 쌍을 맞추면 게임이 완료되며, 점수와 정확도, 콤보, 랭크를 실시간으로 확인할 수 있습니다.

## 주요 기능

- 8쌍, 총 16장의 동물 카드로 진행되는 메모리 매칭 퍼즐
- 카드 선택 횟수, 실수 횟수, 진행 시간, 완성 쌍 수 표시
- 정확도와 콤보에 따라 변하는 점수 시스템
- S+부터 C까지의 랭크 평가
- 숨겨진 카드를 잠시 보여주는 힌트 기능
- 최고 점수를 브라우저 `localStorage`에 저장
- 키보드 방향키로 카드 이동이 가능한 접근성 지원

## 게임 방법

1. 카드를 한 장 선택해 동물을 확인합니다.
2. 두 번째 카드를 선택해 같은 동물인지 맞춰봅니다.
3. 같은 동물이면 카드가 열린 상태로 유지되고 점수가 올라갑니다.
4. 다른 동물이면 카드가 다시 닫히며 실수 횟수가 증가합니다.
5. 모든 8쌍을 맞추면 최종 점수와 기록이 저장됩니다.

## 점수 규칙

- 기본 매칭 점수에 시간 보너스, 시도 효율 보너스, 콤보 보너스가 더해집니다.
- 정확도가 80% 이상이면 추가 보너스를 받을 수 있습니다.
- 잘못 맞추면 점수가 차감되고 콤보가 초기화됩니다.
- 힌트를 사용하면 점수가 차감됩니다.
- 실수 없이 빠르게 완료하면 퍼펙트 보너스를 받을 수 있습니다.

## 실행 방법

별도의 빌드 과정이 필요하지 않습니다.

```bash
git clone https://github.com/사용자명/ChatGPT-Puzzle.git
cd ChatGPT-Puzzle
```

그다음 `index.html` 파일을 브라우저에서 열면 게임을 실행할 수 있습니다.

로컬 서버로 실행하고 싶다면 아래처럼 간단한 정적 서버를 사용할 수 있습니다.

```bash
python -m http.server 8000
```

브라우저에서 `http://localhost:8000`으로 접속합니다.

## 프로젝트 구조

```text
.
├── index.html          # 게임 화면 마크업ㅍㅍ
├── css/
│   └── styles.css      # 레이아웃, 카드 애니메이션, 반응형 스타일
├── js/
│   ├── app.js          # DOM 요소 연결 및 게임 초기화
│   ├── board.js        # 카드 보드 렌더링과 키보드 포커스 이동
│   ├── game.js         # 게임 상태, 점수, 타이머, 힌트 로직
│   └── utiles.js       # 상수, 카드 데이터, 기록 저장 유틸리티
└── assets/
    └── images/         # 파비콘 이미지
```

## 기록 저장

최고 점수는 브라우저의 `localStorage`에 저장됩니다. 같은 브라우저에서 다시 접속하면 이전 최고 점수를 확인할 수 있습니다.
