# 프로젝트: 웹 기반 동물 퍼즐 게임

## 목표

- 브라우저에서 실행되는 4x4 동물 짝 맞추기 퍼즐 게임을 만든다.
- 초보자도 코드를 읽고 수정할 수 있도록 단순한 HTML, CSS, JavaScript 구조를 유지한다.
- 사용자는 카드 두 장을 열어 같은 동물 그림을 찾고, 모든 쌍을 맞추면 게임을 완료한다.

## 기능

- 4x4 카드 보드 표시
- 같은 동물 카드 2장씩 총 8쌍 구성
- 카드 클릭 이벤트 처리
- 같은 동물인지 짝 맞추기 판정
- 맞춘 카드는 열린 상태로 고정
- 시도 횟수, 실수 횟수, 시간, 점수, 정확도, 콤보, 진행률 표시
- 힌트 기능 제공
- 최고 기록 저장
- 게임 완료 조건 판정

## 기술

- 순수 HTML, CSS, JavaScript
- 별도 빌드 도구 없이 브라우저에서 바로 실행
- 핵심 파일은 `index.html`, `style.css`, `script.js`로 분리

## 대상

- 코딩 시니어 개발자

## 폴더 구조

```text
puzzle-game/
├─ index.html
├─ css/
│  └─ styles.css
├─ js/

│  └─ utiles.js -- 공통함수

│  └─ sounds/

```
